# Lara-Agente de Cobranca

Projeto base para o agente de cobranca Lara, construido com Vite + React + TypeScript.

## Requisitos

- Node.js 20+
- npm (ou Bun, se preferir)

## Como executar

```bash
npm install
npm run dev
```

Aplicacao local: `http://localhost:5173`

## Scripts uteis

- `npm run dev`: inicia ambiente local
- `npm run build`: gera build de producao
- `npm run preview`: sobe build local para validacao
- `npm run test`: executa testes com Vitest
- `npm run lint`: executa ESLint
